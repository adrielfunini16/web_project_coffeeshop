# Triple Espresso

Coffee Shop – TripleTen Project

Este projeto faz parte da Sprint de HTML e CSS Avançado da TripleTen. A proposta é recriar a página de uma cafeteria fictícia, seguindo com precisão o layout fornecido no brief. O foco principal é a organização dos blocos, o uso correto da metodologia BEM e a atenção aos detalhes visuais.

Descrição do projeto

A estrutura do site foi construída completamente em HTML e estilizada com CSS puro. O objetivo foi reproduzir fielmente o design original, respeitando espaçamentos, tipografia, cores, posicionamento dos elementos e comportamento dos blocos.
O projeto inclui:
• Header com navegação
• Conteúdo principal com formulário e seções informativas
• Footer completo com logotipo, links sociais e copyright
• Uso de Flexbox e layout organizado em blocos BEM
• Imagens otimizadas e estrutura de arquivos organizada

Tecnologias utilizadas
• HTML5
• CSS3
• Metodologia BEM
• Flexbox
• DevTools para inspeção e validação de layout

Melhorias planejadas

Enquanto desenvolvia o projeto, surgiram algumas ideias para deixar a experiência mais agradável e moderna. Estas melhorias não fazem parte do escopo da Sprint, mas serão implementadas posteriormente na versão de portfólio:
• Rolagem suave (scroll-behavior: smooth;) para navegação interna mais fluida
• Efeitos de hover nos links do menu e do rodapé, trazendo mais vida e feedback visual
• Aprimoramento da responsividade, garantindo que o layout se adapte melhor a diferentes resoluções e dispositivos
• Ajustes finos de tipografia e microespaçamentos para entregar uma versão ainda mais refinada

Essas melhorias serão adicionadas após a aprovação do projeto oficial, mantendo a versão enviada fiel ao brief, como solicitado na Sprint.
